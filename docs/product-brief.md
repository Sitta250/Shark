# Shark — Product Brief

## Overview

Shark is an AI-powered platform that helps users turn business ideas into:
- a viability assessment
- a structured business strategy
- a step-by-step execution plan

It combines analysis + execution into a single system.

---

## Problem

Most aspiring founders:
- don’t know if their idea is worth pursuing
- waste time building before validating
- lack structured guidance
- don’t know what to do next

---

## Solution

Shark provides:
1. **Decision clarity** — is this idea worth it?
2. **Structured insight** — market, competitors, risks
3. **Execution plan** — what to do next

---

## Core Value

Shark is a:
- decision engine
- business analyst
- execution planner

Not a chatbot.

---

## Target Users

- solo founders
- indie hackers
- early-stage builders
- non-technical founders
- consultants validating ideas

---

## Key Use Cases

- validate a new business idea
- compare multiple ideas
- plan MVP and go-to-market
- identify risks before building
- organize execution in one place

---

## Core Flow

1. User creates project (idea)
2. Completes intake
3. Shark generates report
4. User reviews insights
5. User works inside execution workspace

---

## Differentiation

- structured output (not chat)
- decision-first (not inspiration)
- execution-focused (not just analysis)
- workspace-driven retention

---

## Success Metric (v1)

- user completes intake → gets report → creates at least 1 task in workspace

---

## Anti-Goals

- not a general AI chat tool
- not a full CRM
- not a no-code builder
- not a pitch deck generator (v1)
